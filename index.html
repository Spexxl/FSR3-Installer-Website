FAQ Frequent Questions

Why does Windows Defender report as a Trojan?

There are several reasons for this but the main ones are: Running regedit in the application code, dll manipulation and installing the asi loader which is necessary for the mod to work. There's nothing to worry about, this tool is completely safe and free from trojans, malware or anything like that.

Is it compatible with Linux and Mac OS?

Yes, on the Linux system it is necessary to run it through Lutris and on Mac OS through Mac OS Crossover.

Is it necessary to select the specific game directory?

"No, you should only select the first directory. This instruction specifically applies to games that are included in the provided list. Ensure you do not select any additional directories beyond the first one for these listed games."
